S105N1 Exercices 1 to 5

Guía para ejecución de los programas:

Importante: para la correcta compilación y ejecución del programa es necesario contar con ApachaCommonsLang en su versión 3 o posterior.

Compilación
- Acceder al Terminal, CMD o intérprete de la línea de comandos correspondiente.
- Utilizar el comando "cd" para seleccionar el directorio que contenga nuestros archivos.
I.e.:  C:\Users\user1>cd big_folder\program_folder

- Ejecutar el comando javac para compilar señalando entrecomillado la ruta del archivo .jar Apache Commons Lang, seguido de las clases, situando la última la que contenga el main.
I.e.: C:\Users\user1\cd big_folder\program_folder>javac -cp "C:Users\user1\jar\commons-lang3.0.jar" Alfabetico.java Principal.java

Ejecución
- El programa mediante el comando "java", señalando la Class Path del archivo .jar y la clase que contiene el main.
I.e.: C:\Users\user1\cd big_folder\program_folder>javac -cp "C:Users\user1\jar\commons-lang3.0.jar" Principal.java

Ejercicios
1. Introduce por teclado la ruta de un archivo de texto para que se muestre su contenido ordenado alfabéticamente
2. Introduce por teclado la ruta de un directorio para lista el árbol de directorios con el contenido de todos sus niveles y mostrarlo por pantalla.
3. Introduce por teclado la ruta de un directorio para listar un árbol de directorios con el contenido de todos sus niveles y almacenarlo en un archivo .txt.
4. Introduce por teclado la ruta de un archivo .txt para mostrar su contenido en la consola.
5. Introduce por teclado un directorio donde se serializará un objeto. Seguidamente se mostrará por pantalla el objecto deserializado.
